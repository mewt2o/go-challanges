Master Go programming patterns through hands-on practice.

## 
Learning Go effectively requires more than just reading documentation. This repository provides structured exercises covering various Go programming patterns—from concurrency to design patterns, testing, and performance optimization. Learn by doing, not just reading.

## Getting Started

### How to Use
1. **Clone the repository**
```bash
git clone https://github.com/ngobam/go-challenges.git
cd go-challenges
```

2. **Navigate to a challenge**
```bash
cd 0-stock-aggregator
```

3. **Solve the challenge**
   - Only edit `main.go` to solve the problem
   - Do not modify any other files
